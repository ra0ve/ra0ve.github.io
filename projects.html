Operating Systems

March 2020 - June 2020

  • Extended the xv6 kernel by modifying core process control functionality: added support for exit(int status), wait(int *status), and implemented a new waitpid() system call with PID-specific blocking behavior.
  • Replaced xv6's round-robin scheduler with a priority-based one, adding dynamic priority adjustment, starvation prevention (via aging), and optional features like priority donation and process runtime metrics tracking.
  • Redesigned xv6's user-space memory layout to separate heap and stack regions, relocated the stack to the top of the address space, and implemented automatic backward stack growth using page fault handling.
  • Built shared memory support in xv6 by implementing shm_open() and shm_close() system calls; allowed inter-process page sharing via reference counting, page table manipulation, and user-level spinlocks.
